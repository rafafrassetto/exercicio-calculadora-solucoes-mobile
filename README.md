# Calculadora React Native - Exercício Acadêmico

Este projeto consiste em uma calculadora simples desenvolvida com React Native e Expo, utilizando a biblioteca React Native Paper para a interface do usuário.

## Descrição

O aplicativo foi desenvolvido como parte de uma atividade acadêmica para compreensão de componentes, frameworks visuais e Hooks no React Native. A interface apresenta uma temática estética baseada em hieróglifos egípcios, onde tanto os botões quanto o display utilizam caracteres simbólicos para representar números e operações.

## Funcionalidades

- Operações matemáticas básicas: adição, subtração, multiplicação e divisão.
- Interface customizada com React Native Paper.
- Tradução em tempo real de valores numéricos para simbologia de hieróglifos.

## Instruções de Execução

1. Instalar as dependências:
   ```bash
   npm install
   ```

2. Iniciar o projeto:
   ```bash
   npx expo start
   ```

## Tecnologias Utilizadas

- React Native
- Expo
- React Native Paper
- Styled Components (estilização interna)
